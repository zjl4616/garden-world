# 欧阳的花园世界 🌱

一个简单的网页版花园种植游戏，纯前端实现，无需后端。

## 🎮 游戏特色

### 核心玩法
- **种植系统**：5种不同的植物可供种植
- **生长系统**：植物随时间生长，从幼苗到成熟
- **收获系统**：成熟后可收获获得金币
- **商店系统**：用金币购买新的种子
- **等级系统**：通过种植获得经验升级

### 植物种类
1. **向日葵** 🌻 - 基础植物，生长快
2. **胡萝卜** 🥕 - 营养蔬菜，中等收益
3. **玫瑰** 🌹 - 美丽花朵，较高收益
4. **仙人掌** 🌵 - 耐旱植物，稳定收益
5. **苹果树** 🍎 - 高级植物，高收益但生长慢
6. **欧阳花** 🌸 - 神秘花朵，收获时有额外奖励

## 🚀 快速开始

### 在线访问
将文件部署到任意静态网站托管服务即可：

1. **GitHub Pages**
2. **Vercel**
3. **Netlify**
4. **任何静态服务器**

### 本地运行
```bash
# 使用 Python 简单服务器
python3 -m http.server 8000

# 或使用 Node.js
npx serve .
```

然后在浏览器打开：http://localhost:8000

## 🛠️ 技术特点

### 前端技术
- **纯 HTML/CSS/JS** - 无框架依赖
- **响应式设计** - 适配手机和电脑
- **本地存储** - 使用 localStorage 保存进度
- **PWA 支持** - 可安装为桌面/手机应用
- **离线可用** - 无网络也可玩

### 游戏特性
- **自动保存** - 每30秒自动保存进度
- **实时生长** - 植物随时间真实生长
- **视觉反馈** - 丰富的动画和效果
- **进度恢复** - 关闭浏览器后进度不丢失

## 🎯 游戏指南

### 基础操作
1. **选择种子**：在商店点击种子选择
2. **种植**：点击空地种植已选择的种子
3. **浇水**：加速植物生长
4. **收获**：植物成熟后点击收获获得金币
5. **升级**：用金币购买更高级的种子

### 工具功能
- **全部浇水**：一键给所有植物浇水
- **全部收获**：一键收获所有成熟植物
- **清理空地**：一键清理所有空地

### 游戏管理
- **保存游戏**：手动保存当前进度
- **加载游戏**：恢复上次保存的进度
- **重置游戏**：清空所有进度重新开始

## 🔧 扩展开发

### 添加新植物
在 `game.js` 的 `seeds` 数组中添加新植物：

```javascript
{
    id: 6,
    name: '欧阳花🌸',
    price: 45,
    icon: '🌸',
    growthTime: 130,
    value: 85,
    level: 3,
    color: '#d53f8c',
    description: '传说中的神秘花朵，开花时会有特殊效果，收获时获得额外金币和经验'
}
```

### 修改游戏参数
- **花园大小**：修改 `gardenSize` 变量
- **初始金币**：修改构造函数中的 `coins` 值
- **升级经验**：修改 `addExperience` 方法中的计算公式

### 添加新功能
- **装饰系统**：添加花园装饰品
- **好友系统**：添加好友互动
- **成就系统**：添加成就和奖励
- **天气系统**：添加天气影响生长

## 📱 PWA 功能

游戏支持 PWA（渐进式网页应用），可以：
1. **添加到主屏幕**（手机/电脑）
2. **离线运行**（无需网络）
3. **全屏体验**（无浏览器界面）
4. **推送通知**（需要 Service Worker）

## 🎨 美术资源

游戏使用 Emoji 作为植物图标，无需额外图片资源。如需替换：
1. 准备 SVG/PNG 图标
2. 替换对应的 Emoji 字符
3. 调整样式中的图标大小和颜色
